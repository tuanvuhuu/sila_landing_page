# SILA English Center - Landing Page

Trang web landing page hiện đại và chuyên nghiệp cho SILA English Center, trung tâm tiếng Anh dành cho trẻ em và thanh thiếu niên.

## 🎯 Tính Năng Chính

- **Thiết kế Responsive**: Tương thích hoàn hảo với tất cả các thiết bị (desktop, tablet, mobile)
- **Màu sắc SILA**: Sử dụng bảng màu chính thức của SILA (xanh dương đậm, xanh dương nhạt, vàng gold)
- **Hiệu ứng Động**: Các hiệu ứng hover và animation mượt mà
- **SEO Tối ưu**: Cấu trúc HTML semantic
- **Tốc độ nhanh**: Giảm thiểu JavaScript, CSS tối ưu
- **Dễ tùy chỉnh**: Code sạch, dễ bảo trì và mở rộng

## 📋 Các Phần Chính

1. **Navigation Bar** - Menu điều hướng sticky với logo SILA
2. **Hero Section** - Phần giới thiệu nổi bật với CTA button
3. **Programs Section** - Hiển thị các khóa học chính
4. **About Section** - Thông tin về SILA và số liệu thống kê
5. **Features Section** - Các ưu điểm nổi bật
6. **Testimonials** - Nhận xét từ phụ huynh
7. **Contact Section** - Biểu mẫu liên hệ
8. **Footer** - Thông tin liên hệ và social links

## 🚀 Cách Sử Dụng

### Chạy Locally
1. Tải toàn bộ dự án
2. Mở file `index.html` trong trình duyệt web
3. Hoặc sử dụng một local server:
   ```bash
   # Sử dụng Python
   python -m http.server 8000
   
   # Hoặc Node.js (với http-server)
   npx http-server
   ```
4. Truy cập `http://localhost:8000` trong trình duyệt

### Tùy Chỉnh
- Thay đổi thông tin công ty trong `index.html`
- Cập nhật màu sắc trong biến CSS trong `styles.css` (phần `:root`)
- Thêm các chức năng mới trong `script.js`
- Thay đổi logo bằng cách thay thế file `logo.png`
